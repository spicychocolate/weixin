# java微信公众号开发
tips:搭建框架SSM
### 基本功能包括
  * 消息推送与接收（文本消息、图文消息）
  * 用户信息网页授权
  * 发送模板消息
  * 扫码事件
  * 获取地理位置
  * 微信jssdk
